Установка Сфинкса на Windows не вызывает никаких трудностей, просто загрузил архив, распаковал.

Потом, в соответствии с документацией из раздела Getting started on Windows (http://sphinxsearch.com/docs/sphinx3.html#getting-started-on-windows)
 из командной строки Windows запустил sphinxd.exe 

Затем из другой сессии командной строки Windows продолжил по разделу Running queries via MySQL shell (http://sphinxsearch.com/docs/sphinx3.html#getting-started-on-windows)

Вызвал команду "C:\Program Files\MySQL\MySQL Server 8.0\bin\mysql.exe" -h127.0.0.1 -P9306

Далее вызывал команды (я так понимаю, языка SphinxQL) из этого же раздела.

При таком подходе индексация происходит в реальном времени, что понижает производительность Сфинкса.

Чтобы этого избежать и идти дальше (в частности, повторить пример из статьи IBM, показанной на уроке) нужно будет создать файл конфигурации sphinx.conf и, затем,
произвести индексацию приложением indexer.exe
